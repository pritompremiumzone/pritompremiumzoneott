HOW TO UPLOAD TO GITHUB PAGES

1. Unzip this file.
2. Open the ppzott.shop folder.
3. Upload these items directly to your GitHub repository ROOT:
   - index.html
   - payment.html
   - style.css
   - script.js
   - assets folder

IMPORTANT:
- Do NOT upload only the ZIP file.
- Do NOT upload only index.html.
- The assets folder must stay beside index.html, like this:

index.html
payment.html
style.css
script.js
assets/logos/youtube.png
assets/logos/netflix.png
assets/payment/bkash.png

If images are not showing, the assets folder is missing or not in the same root location as index.html.
